# Ruby Training
このリポジトリは、Ruby の基礎から応用までを体系的に学ぶための学習用リポジトリです。
1行ずつ丁寧に言語化・解説しながら理解を深めるたり、コードの読み書きができるようになることを目的としています。

## 🎯 学習目的
- Ruby の文法・基本メソッドを確実に理解する。
- コードを「読める」「説明できる」「書ける」力を身に付ける。
- 後から見返して復習やリファクタリング練習ができるようにする。
